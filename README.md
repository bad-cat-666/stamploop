# StampLoop MVP 🚀

A lightweight, cloud-native Minimum Viable Product (MVP) designed to digitize traditional stamp cards for local businesses. 

## 🌟 System Architecture
This dual-terminal system operates in real-time without the need for manual page refreshes:
*   **Customer Terminal (`index.html`):** Displays real-time points and a unique static QR code for user identification.
*   **Admin Terminal (`admin.html`):** The merchant dashboard used to execute transactions (adding points) directly to the cloud.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, Vanilla JavaScript, CSS
*   **Backend & Database:** Firebase (Cloud Firestore)
*   **Real-time Communication:** Firestore `onSnapshot` listeners
*   **Deployment:** GitHub Pages
